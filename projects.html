Cow Gazer 🤠

A cattle tracking system used by farmers in Livno, Bosnia and Herzegovina 🐮. Working on it with people from Isaac code.

The GitHub repository for the object detection part of the solution is here. We are writing a paper on the process and implementing a cost-effective modular solution using an edge, a fully autonomous internet-to-server connection, a platform for monitoring the weight and vision data with future predictions.

Interactive Problem Repository ♣

This is a random problem generator with a twist. After seeing things very similar to this, for example: Marabot, Trivial and AMC trainer and noticing that these are mainly for AMC/AIME and pre-olympiad level math, I wanted to make something for IMO participants and people near that level. The most important aspect is the MOHS hardness which is determined by user feedback. This is in work in progress for general public use.
